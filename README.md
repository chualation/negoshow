# Ne-goshow Talipapa Utility

## Requirements
- Node.js 18 or newer
- npm 9 or newer

## Run locally
```bash
npm install
npm run dev
```
Open the local URL printed by Vite (usually http://localhost:5173).

## Production build
```bash
npm run build
npm run preview
```
